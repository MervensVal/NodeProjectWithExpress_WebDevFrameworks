(Get, Post, Patch, & Delete works with Postman)
GET /users http://localhost:3000/html/get.html Will return a list of any user in the database (as a JSON array) with all their properties
POST /users http://localhost:3000/html/post.html Will allow you to Update (change) any data (except the ID) for an existing User.
  • UserID
  • Fist Name
  • Last Name
  • Email Address
  • Password
If the user is successfully created it will return the user entity along with a 201 (Created Status)
GET /Users/{userId} None Will return a specific User ID (the user ID provided in the URL) with all its properties if it exists.
If the User ID is not found it will return an error 404 (Not Found) with an appropriate message
PATCH /Users/{userId} None Will allow you to Update (change) any data (except the ID) for an existing User.
If the User isn’t found it will return an error (same as above 404 not found)
DELETE /User/{ID} None Will Delete a user (if it exists) if it doesn’t exist it will return an Error with a status code 404 (Not found)
